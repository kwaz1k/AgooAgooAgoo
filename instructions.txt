# Инструкция по SQL запросам и работе с LibreOffice

## SQL запросы

### Основные SQL запросы:

1. **SELECT** - для извлечения данных из базы данных:
   ```sql
   SELECT column1, column2 FROM table_name WHERE condition;
   ```

2. **INSERT** - для добавления новых записей:
   ```sql
   INSERT INTO table_name (column1, column2) VALUES (value1, value2);
   ```

3. **UPDATE** - для изменения существующих записей:
   ```sql
   UPDATE table_name SET column1 = value1 WHERE condition;
   ```

4. **DELETE** - для удаления записей:
   ```sql
   DELETE FROM table_name WHERE condition;
   ```

5. **JOIN** - для объединения таблиц:
   ```sql
   SELECT * FROM table1 JOIN table2 ON table1.column = table2.column;
   ```

## Работа с LibreOffice

### LibreOffice Calc (Электронные таблицы):

1. **Запуск**: Откройте LibreOffice Calc через меню Пуск или ярлык.

2. **Импорт данных из SQL**:
   - Перейдите в меню "Данные" > "Новый запрос SQL"
   - Настройте соединение с базой данных
   - Введите SQL запрос
   - Нажмите "Выполнить"

3. **Экспорт данных в SQL**:
   - Выделите диапазон данных
   - Перейдите в меню "Файл" > "Экспорт"
   - Выберите формат SQL или CSV

4. **Формулы и функции**:
   - Введите "=" для начала формулы
   - Используйте функции SUM(), AVERAGE(), COUNT() и др.

### LibreOffice Base (Базы данных):

1. **Создание новой базы данных**:
   - Запустите LibreOffice Base
   - Выберите "Создать новую базу данных"
   - Следуйте указаниям мастера

2. **Подключение к существующей базе**:
   - Запустите LibreOffice Base
   - Выберите "Подключиться к существующей базе данных"
   - Выберите тип базы данных (MySQL, PostgreSQL и т.д.)
   - Введите параметры подключения

3. **Создание SQL запросов**:
   - Откройте базу данных в LibreOffice Base
   - Перейдите на вкладку "Запросы"
   - Выберите "Создать запрос в режиме SQL"
   - Введите SQL запрос
   - Сохраните и выполните запрос

### LibreOffice Writer (Текстовый процессор):

1. **Вставка данных из базы**:
   - Перейдите в меню "Вставка" > "Поля" > "Другое"
   - Выберите вкладку "База данных"
   - Выберите таблицу или запрос
   - Выберите поля для вставки

2. **Создание формы для работы с данными**:
   - Перейдите в меню "Форма" > "Дизайн формы"
   - Добавьте необходимые элементы управления
   - Свяжите их с полями базы данных

## Полезные советы:

1. Всегда делайте резервное копирование данных перед выполнением UPDATE или DELETE запросов.
2. Используйте параметризованные запросы для предотвращения SQL-инъекций.
3. Оптимизируйте сложные запросы для повышения производительности.
4. Создавайте индексы для часто используемых полей в WHERE условиях.
5. Регулярно выполняйте VACUUM (для PostgreSQL) или OPTIMIZE TABLE (для MySQL) для поддержания производительности.
